sequenceDiagram
  participant YT as youtube.com/watch
  participant CS as Content script
  participant BG as Background
  participant API as Backend API

  YT->>CS: Navigation / debounced check
  CS->>BG: GET_ACTIVE_RULES
  BG-->>CS: rules in current schedule window

  loop each active rule
    CS->>BG: CHECK_METADATA (+ metadata, url, description)
    BG->>API: POST /check-metadata
    API-->>BG: matches, confidence, reasoning
    BG-->>CS: result
    alt matches and confidence ≥ threshold
      CS->>BG: REDIRECT_TO_BLOCKED
      BG->>YT: Navigate to blocked page
    end
  end
